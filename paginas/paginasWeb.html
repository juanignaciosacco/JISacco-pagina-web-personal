Páginas web

Proceso

1 - Conociendo al cliente

Aqui se arreglan entrevistas para conocernos y ver tus necesidades.

2 - Compra de dominio

Compramos el dominio de su pagina web y el servicio de hosting

3 - Plasmando tu idea

Se comienza a diseñar la pagina web.

4 - Presentacion del bosquejo

Presetamos nuestro prototipo de pagina web segun lo conversado en la primer entrevista y las necesidades mencionadas.

5 - Cargamos la pagina web al servidor

Se carga la pagina web a un servidor de prueba para que puedas acceder desde tus dispositivos y tengas la posibilidad de modificar algun contenido o diseño antes de asociarla al dominio.

6 - Subida del sitio al dominio

Este es el utlimo paso en el que ya asociamos la pagina web a tu dominio y queda pronta para que todo el mundo pueda acceder
